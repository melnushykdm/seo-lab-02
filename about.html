Про проект

Цей сайт створено як навчальний проєкт для виконання лабораторної роботи з SEO. Мета — навчитися аналізувати метадані сторінок, структуру заголовків та технічні параметри сайту.

Сайт навмисно містить кілька типових SEO-помилок (наприклад, відсутній meta description на деяких сторінках), щоб можна було їх знайти та описати у звіті.

Що ми перевіряємо

  • Title та meta description
  • H1–H6 заголовки
  • Canonical
  • Внутрішні посилання
  • Дублікати метаданих
